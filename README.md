# Agentforce AI Service Agent
AI-powered customer service agent built with **Salesforce Agentforce Builder**

---

## Project Overview

This project demonstrates how to build an **AI Service Agent using Salesforce Agentforce** that can automate customer support interactions for a resort booking system.

The agent helps guests:

• Learn about resort experiences  
• Validate customer identity  
• Check session availability  
• Book activities automatically  

The solution combines **Agentforce AI reasoning, Salesforce Flow automation, and CRM data** to create an intelligent service workflow.

---

## Demo Use Case

Customer asks:

> "Can you tell me about the Underground Cave Exploration?"

The AI agent will:

1. Verify the customer identity  
2. Retrieve experience details  
3. Check available sessions  
4. Book the experience  

All actions happen **automatically through Salesforce CRM data and flows.**
