# Intent Detection

## Overview

Intent detection is the process of analyzing a customer's message to determine what the user wants to accomplish.

The Agentforce AI model examines the message and identifies the user's goal.

## Example

Customer message:

Tell me about the Underground Cave Exploration.

Detected intent:

Request information about a resort experience.

## Supported Intents

The agent supports several types of requests:

• Experience information requests  
• Session availability inquiries  
• Experience booking requests  

Intent detection helps the agent determine which topic should handle the request.
