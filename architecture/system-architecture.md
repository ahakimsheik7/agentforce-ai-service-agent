# Agentforce AI Service Agent – System Architecture

## Overview

This project demonstrates the architecture of an AI-powered customer service agent built using **Salesforce Agentforce Builder**.

The agent assists Coral Cloud Resort guests with:

- Learning about resort experiences
- Validating customer identity
- Checking available activity sessions
- Booking resort experiences

The system combines **Agentforce AI reasoning, Salesforce Flow automation, and CRM data** to create an intelligent conversational service agent.

---

# High-Level Architecture

The system architecture consists of six main layers:

1. Customer Interaction Layer
2. Messaging Channel Layer
3. Agentforce AI Layer
4. Topic Reasoning Layer
5. Action Execution Layer
6. Salesforce Data Layer
