# Agentforce AI Service Agent – Agent Reasoning Architecture

## Overview

This document explains how the **Agentforce Service Agent** determines what actions to take during a customer conversation.

Agentforce uses a **topic-based reasoning system** combined with instructions and actions to determine how the AI agent should respond to user requests.

The reasoning process allows the agent to:

• Understand the user's intent  
• Select the correct topic  
• Determine which actions to execute  
• Generate a clear and helpful response  

---

# Agent Reasoning Flow

The agent follows a structured reasoning process when responding to customer messages.

Customer Input  
↓  
Intent Detection  
↓  
Topic Selection  
↓  
Instruction Evaluation  
↓  
Action Execution  
↓  
Response Generation
